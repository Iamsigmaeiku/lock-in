# 快速開始指南

## 運行應用

確保你已經安裝了 Flutter SDK，然後在專案目錄執行：

```bash
# 安裝依賴
flutter pub get

# 運行應用（iOS 模擬器）
flutter run

# 或選擇設備
flutter devices
flutter run -d <device-id>
```

## 首次使用

### 1. 開始你的第一個番茄鐘
- 打開應用，你會看到主頁面的圓形計時器
- 點擊「開始」按鈕開始 25 分鐘的工作時段
- 計時器會自動倒數，完成時會有通知提醒

### 2. 創建任務
- 點擊底部導航欄的「任務」
- 點擊右下角的「+」按鈕
- 輸入任務名稱（例如：「寫報告」）
- 設定預估需要幾個番茄鐘（例如：3 個）
- 點擊「新增」

### 3. 關聯任務到計時器
- 回到主頁面
- 在「當前任務」卡片中點擊「選擇任務」
- 從列表中選擇你剛創建的任務
- 開始計時，完成後會自動記錄進度

### 4. 查看統計
- 點擊底部導航欄的「統計」
- 查看今日完成的番茄鐘數和專注時長
- 週統計柱狀圖顯示每天的表現
- 熱力圖展示最近 30 天的專注習慣

### 5. 個性化設定
- 點擊底部導航欄的「設定」
- 調整工作時長、休息時長等
- 選擇喜歡的主題色彩
- 開啟深色模式

## 功能亮點

### 計時器頁面
- **圓形進度條**：直觀顯示剩餘時間
- **階段標籤**：清楚顯示當前是工作還是休息
- **快速操作**：開始、暫停、重置、跳過
- **任務關聯**：選擇當前要做的任務

### 任務頁面
- **分頁顯示**：進行中和已完成任務分開
- **進度追蹤**：顯示完成進度條和百分比
- **拖拽排序**：長按並拖動調整順序
- **滑動刪除**：左滑快速刪除任務

### 統計頁面
- **今日概覽**：快速查看今日數據
- **週統計**：柱狀圖展示本週表現
- **熱力圖**：類似 GitHub 的活動熱力圖
- **總計數據**：累計完成的番茄鐘數

### 設定頁面
- **計時器設定**：自訂所有時間長度
- **通知設定**：控制通知和音效
- **外觀設定**：5 種主題色 + 深色模式
- **數據管理**：清除或重置數據

## 使用技巧

### 提高效率
1. **早上規劃**：每天開始前創建當天的任務列表
2. **合理預估**：根據任務複雜度預估番茄鐘數
3. **專注工作**：工作時段全神貫注，不要分心
4. **充分休息**：休息時段離開螢幕，真正放鬆

### 養成習慣
1. **每日目標**：設定每日完成番茄鐘的目標（例如 8 個）
2. **查看統計**：定期查看統計頁面，了解自己的習慣
3. **持續改進**：根據統計數據調整時間設定

### 進階用法
1. **長休息利用**：長休息時可以做些伸展運動
2. **任務分解**：把大任務分解成多個小任務
3. **主題切換**：根據時段選擇不同的主題色彩

## 常見問題

**Q: 可以自訂時間嗎？**  
A: 可以！在設定頁面調整工作時長、短休息和長休息時長。

**Q: 數據會丟失嗎？**  
A: 不會，所有數據都儲存在本地，除非你主動清除。

**Q: 通知不響應怎麼辦？**  
A: 請確保在系統設定中允許應用發送通知。

**Q: 可以中途暫停嗎？**  
A: 可以！點擊暫停按鈕，需要時再繼續。

**Q: 如何刪除任務？**  
A: 在任務頁面左滑任務卡片即可刪除。
